AI Mouse Using Hand Gesture

This Virtual Mouse allows users to control their computer cursor using hand gestures detected through a webcam. Leveraging OpenCV for real-time image processing, Mediapipe for hand tracking, and PyAutoGUI for simulating mouse actions, the system provides a touchless, hygienic, and accessible alternative to traditional input devices.

Voice Controlled Home System

This project develops a Voice Controlled Smart Home System using the ESP-32 microcontroller and 4 relay modules to automate home appliances. User voice commands are converted to text via Speech-to-Text (STT) technology, matched against AIML patterns to determine actions, and executed by triggering relays to turn devices on or off.
